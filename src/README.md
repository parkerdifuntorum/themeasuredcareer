# Job Search Smarter

Updated front-end implementation includes:

- Rebranded app title: **Job Search Smarter**
- Email Digest section with a recipient email input
- No default email address
- Salary range input fields for compensation preference
- Modality buttons for Remote, Hybrid, and On-site
- Expanded industry dropdown
- Location preference text input
- Independent optimization-weight sliders
- Default AI ranking weights when no custom weights are selected
- Scored job results instead of labels

## Run locally

```bash
npm install
npm run dev
```
